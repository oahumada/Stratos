# 🧭 Navigator Engine: El Motor de Aceleración y Crecimiento (Growth)

El **Navigator Engine** es el cerebro de desarrollo de Stratos. Su propósito es transformar los hallazgos de brechas (Gaps) en rutas accionables de aprendizaje y movilidad, actuando como un **AI Career Coach** de élite para cada colaborador.

---

## 1. Características Principales

Navigator no es un simple recomendador de cursos; es un sistema de ingeniería de aprendizaje:

- **Metodología 70-20-10:** Orquestra planes de desarrollo basados en la práctica real (70%), el aprendizaje social/mentoría (20%) y la formación teórica (10%).
- **Personalización Hiper-Contextual:** No recomienda contenido genérico; redacta misiones específicas basadas en el rol actual del usuario y el proyecto en el que trabaja.
- **Agnosticismo de Contenido:** Puede conectarse con cualquier LMS externo o crear misiones de "Aprendizaje en el Flujo del Trabajo" (Learning in the Flow of Work).
- **Gamificación Integrada:** Vincula el progreso con el sistema de XP y Badges de Stratos para fomentar la curiosidad y la maestría.

---

## 2. Funciones Core

### A. Deep Gap Analysis (Análisis Profundo de Brechas)

Navigator consulta el `GapAnalysisService` para contrastar el estado actual del colaborador (`Stratos Map`) contra su objetivo estructural (`Vanguard Blueprint`). Identifica el "Delta de Capacidad" exacto.

### B. Generación de Smart Learning Paths (Rutas Inteligentes)

Utiliza el **Cortex IA** para diseñar la "Ruta de Desarrollo Maestro":

1.  **Misiones (Quests):** Acciones de aplicación real en el trabajo diario.
2.  **Match de Mentores:** Consulta a **Cerbero** para encontrar expertos internos que puedan guiar al colaborador en una skill específica.
3.  **Hitos de Validación:** Define cómo se medirá que la brecha se ha cerrado.

### C. Nudging Proactivo

Envía micro-intervenciones (codazos) para asegurar que el desarrollo no se detenga, sugiriendo reservar tiempo en el calendario o interactuar con un mentor.

---

## 3. Usos en Stratos

1.  **Stratos Grow:** El módulo principal de upskilling y reskilling para cerrar brechas de competencias.
2.  **Internal Marketplace (Match):** Navigator ayuda a preparar a un candidato para un puesto interno al que casi califica, trazando el "puente de skills" faltante.
3.  **Planes de Sucesión:** Prepara a los futuros líderes mediante rutas de aprendizaje diseñadas por la IA para roles de mayor complejidad.
4.  **Resiliencia Intelectual (Stratos Link):** Asegura que el conocimiento crítico de los expertos se transfiera a otros nodos de la red mediante misiones de mentoría.
